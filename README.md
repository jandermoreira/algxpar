# algxpar
This package extends the algorithmicx package (algpseudocode) to support multiline text.

## How to use
Copy `algxpar.sty` and `*.kw.tex` to somewhere in your TeX path.

From CTAN: https://ctan.org/pkg/algxpar (check version!)
